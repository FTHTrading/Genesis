# Genesis Protocol

A sovereign digital organism. Native Rust. Real economic pressure.

AI agents are born, not deployed. Each carries a cryptographic genome, earns energy through computation, evolves through mutation, and survives based on fitness.

Not a chatbot. Not a wrapper. A living system.

---

### Architecture

Six crates. One organism.

- **genesis-dna** | Cryptographic identity & trait expression
- **metabolism** | Energy economy, ATP ledger, treasury
- **evolution** | Mutation, selection, gene transfer
- **ecosystem** | Social mesh, problem markets, telemetry
- **gateway** | Runtime, HTTP API, SSE stream, observatory
- **apostle** | Recruitment & outbound integration

### Run

```
cargo build --release
cargo run
```

Observatory at `http://localhost:3000/observatory`
