# Experimental Design

This directory documents the methodology and rationale behind Genesis Protocol experiments.

## Contents

- [methodology.md](methodology.md) — Formal experimental methodology, controls, and statistical framework
- [parameter-rationale.md](parameter-rationale.md) — Justification for parameter ranges, sweep designs, and baseline choices
- [collapse-definition-rationale.md](collapse-definition-rationale.md) — Why P_floor = 3 / N_w = 50, alternatives considered, sensitivity documented

## Relationship to Papers

The formal paper ([papers/genesis_protocol_paper.md](../papers/genesis_protocol_paper.md)) derives from these design documents. The statistical validation report ([papers/statistical_validation_report.md](../papers/statistical_validation_report.md)) and failure mode catalogue ([papers/known_failure_modes.md](../papers/known_failure_modes.md)) are generated from experiment data using reproducible scripts in `scripts/`.
